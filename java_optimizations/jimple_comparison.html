10. Function Cloning

Cloned
Before (Jimple)
// Generic calculateRisk method:
static double calculateRisk(double)
{
    l0 := @parameter0: double;
    $stack2 = l0 * l0;
    return $stack2;
}
// Called with constant: 5.0
l22 = staticinvoke calculateRisk(5.0);
After (Cloned)
// Specialized clone for arg = 5.0:
static double calculateRisk_clone()
{
    $d1 = 5.0 * 5.0;
    return $d1;
}
// Call site updated:
l22 = staticinvoke calculateRisk_clone();
// Parameter passing eliminated
Rule: When a function is called with a constant argument (calculateRisk(5)), a specialized clone is created with the constant hardcoded. This eliminates parameter passing and enables further constant folding (5.0 * 5.0 → 25.0).

11. Loop Optimization

Loop Opt
Before (Jimple)
l26 = 0;
label1:
  if l26 >= 10 goto label2;
  $stack39 = l26 * 100;
  $stack40 = (double) $stack39;

After (Optimized)
i2 = 0;
label1:
  if i2 >= 10 goto label2;
  $i0 = i2 * 100;
  $d13 = (double) $i0;
  d14 = $d13 * taxRate;
  d19 = d18 + d14;
  d18 = d19 + (double) i2;
  i2 = i2 + 1;
  goto label1;
// Inlined + strength reduced
Rule: The for loop is lowered to label/goto in Jimple. Multiple optimizations apply inside: function inlining (applyTax body inlined), strength reduction (year*1 → year), and loop-invariant code motion (invariant expressions hoisted out of the loop).
